# Baseline Attention Seq2Seq Chatbot

A question-answering chatbot built using Bidirectional LSTM with Attention mechanism, powered by Qwen3 embeddings. This repository contains two versions of the model — a baseline and a Knowledge Base-augmented version.

## Models

### 1. Baseline Attention Seq2Seq 
A standard Seq2Seq chatbot trained on general conversational Q&A data.

- Bidirectional LSTM Encoder
- LSTM Decoder with Attention
- Qwen3-Embedding-0.6B for text embeddings
